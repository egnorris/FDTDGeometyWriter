fdtdgeometrywriter.getPixelMaterial(px)[source]¶

GetPixelMaterial() retrieve the Material of the Current Pixel.

Take in a pixel of the material image and determine based on the RGB value if this pixel is the current material or the background material. White Pixels correspond to background material and Black pixels correspond to Current Material pixels with in-between colors

are returned as 0.5 which is presently unused in the writer algorithm

Parameters:

px -- (Array) Current Pixel of Geometry Image File

Returns:

Material State - Either 0 for vacuum, 1 for current material, or 0.5 otherwise
